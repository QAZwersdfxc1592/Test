# Tetris 테트리스 게임

현대 테트리스 가이드라인을 따르는 클래식 테트리스 웹 게임입니다. 키보드와 터치 조작을 지원하며, 다국어 인터페이스를 내장하고 있습니다.

## 게임 특징

- 🎮 7가지 클래식 블록, 7-bag 랜덤 알고리즘(7번마다 7종류가 모두 한 번씩 등장)
- 🔄 SRS 양방향 회전(시계방향 + 반시계방향) 및 wall kick 시스템
- 📦 Hold 보관 기능(블록당 1회)
- 👻 Ghost 낙하 위치 미리보기
- 📊 점수·레벨·제거 줄 수·최고점수 영속화(localStorage)
- ⚡ 레벨업 시 자동 가속
- 🏆 T-spin 판정 및 B2B 콤보 보너스
- 🔊 Web Audio 합성 효과음(음소거 가능)
- 🌐 다국어:简体中文 / 繁體中文 / English / 日本語 / 한국어(시스템 언어 자동 감지)
- 📱 모바일 터치 조작, 길게 누르면 연속 입력

## 조작법

| 키 | 기능 |
|----|------|
| ← → | 좌우 이동 |
| ↑ / X | 시계방향 회전 |
| Z | 반시계방향 회전 |
| ↓ | 소프트 드롭 |
| 스페이스 | 하드 드롭 |
| C | 블록 보관(Hold) |
| P | 일시정지 / 계속 |

모바일: 회전 버튼은 짧게 누르면 시계방향, 길게 누르면 반시계방향입니다. 좌우/아래 버튼은 길게 누르면 연속 입력됩니다.

## 실행 방법

브라우저에서 `index.html`을 직접 열면 게임을 시작할 수 있습니다. 로컬 서버를 사용할 수도 있습니다:

```bash
python3 -m http.server 8000
```

그 후 http://localhost:8000/ 에 접속하세요.

## 파일 구성

- `index.html` - 페이지 구조 및 UI 요소
- `style.css` - 스타일, 애니메이션, 반응형 레이아웃
- `game.js` - 게임 핵심 로직(보드, 블록, 점수, 효과음 등)
- `i18n.js` - 국제화 모듈(언어 감지, 번역 사전)
