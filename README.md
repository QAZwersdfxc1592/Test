# Tetris 俄罗斯方块游戏

一个经典的俄罗斯方块网页小游戏，支持键盘和触屏操作。

## 游戏特性

- 🎮 7 种经典俄罗斯方块
- 📊 分数、等级、消除行数统计
- 🔮 下一个方块预览
- ⚡ 等级提升，速度加快
- 📱 支持移动端触屏操作

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 直接落底 |
| P | 暂停/继续 |

## 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。

## 文件结构

- `index.html` - 游戏页面结构
- `style.css` - 样式与动画
- `game.js` - 游戏核心逻辑
