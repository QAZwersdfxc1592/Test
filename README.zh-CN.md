# Tetris 俄罗斯方块游戏

一个经典俄罗斯方块网页小游戏,符合现代俄罗斯方块规范,支持键盘与触屏操作,内置多语言界面。

## 游戏特性

- 🎮 7 种经典方块,7-bag 随机算法(每 7 次必出齐 7 种)
- 🔄 SRS 双向旋转(顺时针 + 逆时针)与 wall kick 系统
- 📦 Hold 暂存方块(每个方块仅一次)
- 👻 Ghost 落点预览
- 📊 分数、等级、消除行数、最高分持久化(localStorage)
- ⚡ 等级提升自动加速
- 🏆 T-spin 判定与 B2B 连击加成
- 🔊 Web Audio 合成音效(可静音)
- 🌐 多语言:简体中文 / 繁体中文 / English / 日本語 / 한국어(自动检测系统语言)
- 📱 移动端触屏控制,长按连发

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ / X | 顺时针旋转 |
| Z | 逆时针旋转 |
| ↓ | 加速下落 |
| 空格 | 直接落底(hard drop) |
| C | 暂存方块(hold) |
| P | 暂停 / 继续 |

移动端:旋转按钮短按顺时针、长按逆时针;左右/下按钮支持长按连发。

## 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。也可用本地服务器:

```bash
python3 -m http.server 8000
```

然后访问 http://localhost:8000/

## 文件结构

- `index.html` - 页面结构与 UI 元素
- `style.css` - 样式、动画与响应式布局
- `game.js` - 游戏核心逻辑(棋盘、方块、评分、音效等)
- `i18n.js` - 国际化模块(语言检测、翻译字典)
