# Tetris テトリス

現代テトリスガイドラインに準拠したクラシックなテトリス Web ゲームです。キーボードとタッチ操作に対応し、多言語インターフェースを内蔵しています。

## ゲーム特徴

- 🎮 7 種類のクラシックブロック、7-bag ランダムアルゴリズム(7 回ごとに全 7 種が必ず出現)
- 🔄 SRS 双方向回転(時計回り + 反時計回り)と wall kick システム
- 📦 ホールド機能(1 ブロックにつき 1 回)
- 👻 ゴーストブロック落下位置プレビュー
- 📊 スコア・レベル・消去ライン数・ハイスコア永続化(localStorage)
- ⚡ レベルアップによる自動加速
- 🏆 T-spin 判定と B2B コンボボーナス
- 🔊 Web Audio 合成効果音(ミュート可能)
- 🌐 多言語:简体中文 / 繁體中文 / English / 日本語 / 한국어(システム言語を自動検出)
- 📱 モバイルタッチ操作、長押し連射対応

## 操作方法

| キー | 機能 |
|------|------|
| ← → | 左右移動 |
| ↑ / X | 時計回りに回転 |
| Z | 反時計回りに回転 |
| ↓ | 下移動 |
| スペース | ハードドロップ |
| C | ホールド |
| P | 一時停止 / 再開 |

モバイル:回転ボタンは短押しで時計回り、長押しで反時計回り。左右/下ボタンは長押し連射に対応します。

## 実行方法

ブラウザで `index.html` を直接開けばゲームを開始できます。ローカルサーバーを使うこともできます:

```bash
python3 -m http.server 8000
```

その後 http://localhost:8000/ にアクセスしてください。

## ファイル構成

- `index.html` - ページ構造と UI 要素
- `style.css` - スタイル、アニメーション、レスポンシブレイアウト
- `game.js` - ゲーム核心ロジック(盤面、ブロック、スコア、効果音など)
- `i18n.js` - 国際化モジュール(言語検出、翻訳辞書)
