# Tetris 俄羅斯方塊遊戲

一個經典俄羅斯方塊網頁小遊戲,符合現代俄羅斯方塊規範,支援鍵盤與觸控操作,內建多語言介面。

## 遊戲特性

- 🎮 7 種經典方塊,7-bag 隨機演算法(每 7 次必出齊 7 種)
- 🔄 SRS 雙向旋轉(順時針 + 逆時針)與 wall kick 系統
- 📦 Hold 暫存方塊(每個方塊僅一次)
- 👻 Ghost 落點預覽
- 📊 分數、等級、消除行數、最高分持久化(localStorage)
- ⚡ 等級提升自動加速
- 🏆 T-spin 判定與 B2B 連擊加成
- 🔊 Web Audio 合成音效(可靜音)
- 🌐 多語言:簡體中文 / 繁體中文 / English / 日本語 / 한국어(自動偵測系統語言)
- 📱 行動裝置觸控操作,長按連發

## 操作說明

| 按鍵 | 功能 |
|------|------|
| ← → | 左右移動 |
| ↑ / X | 順時針旋轉 |
| Z | 逆時針旋轉 |
| ↓ | 加速下落 |
| 空格 | 直接落底(hard drop) |
| C | 暫存方塊(hold) |
| P | 暫停 / 繼續 |

行動裝置:旋轉按鈕短按順時針、長按逆時針;左右/下按鈕支援長按連發。

## 執行方式

直接在瀏覽器中開啟 `index.html` 即可開始遊戲。也可用本地伺服器:

```bash
python3 -m http.server 8000
```

然後造訪 http://localhost:8000/

## 檔案結構

- `index.html` - 頁面結構與 UI 元素
- `style.css` - 樣式、動畫與響應式佈局
- `game.js` - 遊戲核心邏輯(棋盤、方塊、計分、音效等)
- `i18n.js` - 國際化模組(語言偵測、翻譯字典)
